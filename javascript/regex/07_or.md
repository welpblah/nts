# regex: or

- Sometimes we need to allow multiple alternatives. We can separate them with a pipe character,Â `|`, pronounced "or".!

```js
/a|b/.test('a');
// true
/a|b/.test('b');
// true
/a|b/.test('c');
// false
/at|og/.test('cat');
// true
/at|og/.test('dog');
// true
/at|og/.test('bat');
// true
/at|og/.test('horse');
// false
```
